Supported platforms: macOS, Linux, Android via Termux

Copy and paste these commands:


	git clone https://github.com/Maiake-ops/iseeu.git
	cd iseeu
	bash start.sh

You will have options for normal I-See-You for laptops or an optimized Termux version.

Dependencies: Python3, cloudflared
